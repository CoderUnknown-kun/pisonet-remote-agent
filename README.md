# Node Remote Agent

A lightweight remote management agent for monitoring and controlling distributed nodes in a device fleet.

## Features

• Remote command execution  
• System monitoring  
• Screen streaming (WebRTC / MJPEG)  
• Auto-update system  
• Firebase backend integration  

## Architecture

Agent runs on each PC and communicates with a cloud backend to receive commands and report system status.

## Tech Stack

Python  
Firebase / Firestore  
Flask  
WebRTC  
Automation scripts

## Use Case

Built for managing multiple internet café / pisonet computers remotely.
